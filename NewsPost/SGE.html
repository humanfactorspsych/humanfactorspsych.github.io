Self-Aware Persona Project

  • 2021.09.24 - 2022.01.24

  • Yoon Kyung Lee
    Seoyeon Bae
    Jisang Park
Description :

HFPsych lab and Human Interface(Prof. Nam Soo Kim) lab from College of Engineering are going to work on a project funded by Smilegate.AI (Smilegate Holdings, "Building self-aware and empathetic conversational agent based on persona").

We will build a dialog dataset for persona-based attractive open-domain chatbots which give out empathetic response and have long-term memory to maintain self-awareness. The dataset will later be used for machine-learning task for empathetic conversation generation.


인간공학심리 연구실과 서울대학교 전기정보공학부 김남수 교수님의 휴먼인터페이스 연구실이 스마일게이트 AI(주식회사 스마일게이트 홀딩스)의 지원을 받아 공동으로 프로젝트를 진행하게 되었습니다.

공감적인 반응을 출력할 수 있으며, 자기 인식 능력을 유지하는 데 필요한 장기 기억을 가질 수 있는 대화 에이전트(챗봇)를 개발하기 위한 대화 데이터셋을 구축하고자 합니다. 해당 데이터셋은 추후 공감적 대화 생성을 위한 머신러닝 과제에 사용될 것입니다.
